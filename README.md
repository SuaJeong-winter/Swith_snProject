## 소개
이 프로젝트는 사용자가 할 일을 관리할 수 있는 **Todo List 웹 애플리케이션**입니다. 간단한 인터페이스를 통해 할 일 목록을 관리할 수 있으며, 데이터를 로컬 스토리지에 저장해 브라우저를 새로고침해도 데이터가 유지됩니다.

## 기능
- 할 일 추가
- 할 일 수정
- 할 일 삭제
- 완료 상태 표시
